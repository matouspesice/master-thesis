\section*{\centering Abstract}\thispagestyle{plain}\label{abstract-en}
This thesis develops a low-latency motion tracking pipeline for gamified knee rehabilitation in virtual reality. The system uses a standard RGB camera and lightweight pose estimation (e.g., RTMPose) to drive real-time VR feedback with a motion-to-display target below 10\,ms. A pilot study evaluates system feel and user engagement in a single-leg balance task within the ``Architect'' game scenario.

\paragraph*{Keywords:} knee rehabilitation, virtual reality, motion tracking, low latency, pose estimation, gamification, balance
